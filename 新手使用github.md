# Github使用经验

## 基本操作：

下载源码不需要登录(点击："绿色code"-"download zip"即可)

Fork按钮是复制到自己的空间(复刻）

**多读readme**

## 新手调参数
可以把代码中所有"required"改为"default=",等号后面的内容按照readme文件填写。这样就可以在自己的pycharm等运行，不用调参数

## 创建自己的仓库:

右上角加号——new——创建一个新仓库

add a readme file按钮勾选，其余默认即可，对于新手来说

然后即可上传文件

github支持在线修改文件


## 删除仓库：
1.点进去要删除的仓库

2.点击设置，并划到最底部

3.点击删除，按照要求做即可


## 互动交流功能:

在一个项目页面里，点击上方的Issues按钮即可。就可以给别人留言

也可以在自己issues里面回复别人，还可以点击close with comment结束自己的话题

点击starred可以点赞


## 搜索小技巧：
awesome➕搜索的关键词，可以搜索到精品

还可以按照一定条件进行**排序**，推荐以最多星标量排序
