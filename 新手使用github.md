*github使用经验*



下载源码不需要登录(绿色code-download zip)即可



Fork按钮是复制到自己的空间



创建自己的仓库:

右上角加号——new——创建一个新仓库

add a readme file按钮勾选，其余默认即可，对于新手来说

然后即可上传文件



github支持在线修改文件



互动交流功能:

在一个项目页面里，点击上方的Issues按钮即可。就可以给别人留言

也可以在自己issues里面回复别人，还可以点击close with comment结束自己的话题

点击starred可以点赞



awesome➕搜索的关键词，可以搜索到精品
